Scrapear

Scrapear, o web scraping, es una manera de obtener información de las páginas web a través de las líneas de código.

Facebook Scraper

El scraperador de Facebook es un framework que nos permite buscar a los usuarios y obtener informacion en formato de diccionarios, que puede ser usada en una base de datos. El Script realizado seleccionaba la informacion que es util para realizar un analisis estadistico de los post para posteriormente realizar un grafico de las interacciones en funcion de la fecha de la publicacion. Lamentablemente este Framework dejo de funcionar

Instagram Scraper

Instagram Scraper es un framework que se apoya de otro framework que se llama Selenium, este es un automatizador de acciones mediante codigo para ser reflejadas en paginas web, es una buena alternativa cuando la pagina web es dinamica.
